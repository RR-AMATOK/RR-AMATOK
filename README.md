## `$ whoami`

I'm an engineer obsessed with making machine learning **faster**, **smarter**, and **closer to the metal**. My playground is the Apple Silicon ecosystem — where unified memory meets GPU compute and the rules of ML infrastructure get rewritten.

When I'm not forking production-grade ML frameworks to squeeze every TFLOP out of an M-series chip, I'm building full-stack platforms that put AI to work in the real world.

**Currently**: Deep in the Apple Silicon ML stack, porting and optimizing industry-standard frameworks for Metal GPU compute.

**Previously**: Data science, NLP research, and enough Kaggle-style competitions to know that 90% of ML is cleaning the data and the other 90% is hyperparameter tuning.

## 🔬 What I'm Building

<table>
<tr>
<td width="50%" valign="top">

### ⚡ CatBoost for Apple Silicon GPU
**Porting one of the world's most powerful gradient boosting libraries to run natively on Apple Silicon.**

CatBoost is a production-grade GBDT engine used across ranking, classification, and regression at massive scale. I'm adapting its core compute kernels to leverage Metal and the unified memory architecture of M-series chips — meaning faster training, lower power draw, and ML that runs where your data lives. No CUDA required.

`C++` `Metal` `Python` `Apple Silicon`

</td>
<td width="50%" valign="top">

### 🧪 Optuna for MLX
**Hyperparameter optimization, reimagined for Apple's ML framework.**

Optuna is the industry standard for automated hyperparameter tuning. I'm building native integration with Apple's MLX framework so the entire train → tune → evaluate loop stays on-device and on-metal. The goal: zero-friction HPO for anyone training models on Apple Silicon.

`Python` `MLX` `Optuna` `Bayesian Optimization`

</td>
</tr>
